## 常见问题（FAQ）

### 1. 无法连接后端服务怎么办？

1. 确认后端服务正在运行（例如 `http://127.0.0.1:8000/health` 可访问）。  
2. 真机测试时不要使用 `127.0.0.1`，请使用你电脑的局域网 IP。  
3. 保证手机和电脑在同一 Wi-Fi。  
4. 使用应用内“自动发现本机地址”按钮重试。

### 2. 打卡后看不到排行更新？

1. 下拉或点击刷新。  
2. 确认网络可用。  
3. 若后端刚启动，等待几秒后再试。

### 3. 风险分数是否代表医疗诊断？

不是。  
分数仅用于健康教育与行为提醒，不构成医疗诊断或治疗建议。

### 4. 我可以删除自己的数据吗？

可以。请通过支持邮箱发送删除请求，并提供必要的信息以便我们定位记录（例如时间段、设备信息等）。

### 5. 忘记了服务器地址怎么办？

请在设置中使用“自动发现本机地址”，或重新填写可访问的 API 地址。
